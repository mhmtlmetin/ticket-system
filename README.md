# Müşteri Talep Sistemi

1. projeyi bilgisayarınıza indirdikten sonra şu komutları çalıştırın:
    npm install
    json-server --watch src/db.json --port 3001
    npm run dev

2. Admin kulllanıcı girişi:
    username: admin
    password: admin123

  Normal kullanıcı girişi:
    username: user
    password: user123
